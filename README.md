# India Socio-Demographic Insights

This project analyzes Indian Census data using Power BI to uncover
state-wise socio-demographic patterns and regional trends.

## Key Insights
- Population distribution across states
- Literacy rate and gender ratio analysis
- Regional disparities using interactive visuals

## Tools & Technologies
- Power BI
- DAX
- Indian Census Data

## How to View
Download the `.pbix` file and open it using Power BI Desktop.
